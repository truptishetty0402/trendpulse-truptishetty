This mini project, TrendPulse, is designed as an end‑to‑end workflow that takes trending stories from an API and transforms them into actionable insights. It begins by fetching raw story data and storing it in JSON format, then processes that JSON into a structured CSV file for easier handling. The CSV is analyzed using pandas to uncover patterns such as top‑scoring stories, category distributions, and the relationship between scores and comments. Finally, the project visualizes these findings with clear charts and a combined dashboard using matplotlib, making trends easy to interpret at a glance. In essence, TrendPulse demonstrates the full cycle of modern data work: collect → process → analyze → visualize, turning raw API data into meaningful knowledge
